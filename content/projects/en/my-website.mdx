## About this Project

This website serves as my digital business card and portfolio. It showcases my projects, my tech stack and provides an easy way to get in touch with me.

## Features

- Multilingual (German & English)
- Dark & Light Mode
- MDX-based blog and projects
- Fully responsive
- SEO optimized
- Hosted on own VPS with Docker and CI/CD

## Tech Stack

The project uses Next.js with the App Router, TailwindCSS v4 for styling and ShadCN/ui for the component library. Animations are implemented with Framer Motion.
